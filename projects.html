Projects

Sign Language Recognition
  • Collected a dataset of images showcasing various hand gestures corresponding to the American Sign Language (ASL) alphabet.
  • Employed three machine learning algorithms - KNN, CNN, and Logistic Regression - to recognize and translate sign language.
  • The scores indicate that CNN performed well in recognizing sign language gestures; The precision was 0.9768, the recall was 0.9745, and the F1 score was 0.9738.
Fake Job Detection
  • Removed invalid data based on proportion of missing values for each variable and proportion of the city that fake jobs concentrated in; used the stopwords package from nltk library to remove common words to extract key information.
  • Selected features based on the distribution, correlation matrix, and statistical method from scipy library such as chi-suqare test; applied these features to appropriate classification models such as KNN, Naive Bayes, Logistic Regression.
Spam Messages Detection
  • Converted the text to lowercase; removed punctuation marks, web links, common words, and other preprocessing methods for filtering.
  • Used LDA and Tf-Idf for text vectorization and applied it to logistic regression; used Doc2Vec model for word embedding and applied it to LSTM model.
  • Evaluated each model by precision and recall, the LSTM performed the best, with a precision of 99% and a recall of 89%.
